A practical step-by-step guide for teachers and managers to set up and use Play Tornado effectively with your students.

‚ö° Quick Start (5 Minutes)

1. Create Roster

Add your students' names

2. Share Class Code

Students join with the code

3. Assign Work

Give quizzes or assignments

4. Play War Game

Students spend points strategically

üë• Step 1: Setting Up Your Class Roster

1

Go to Roster Management

Click "Manage Rosters" on the homepage or go directly to roster.html

2

Create a New Roster

Click "Create New Roster" and fill out:

  • Roster Name: e.g., "Period 3 English" or "Team Alpha"
  • Description: Brief description of the class
  • Target Percentage: How much of the map students need to control to win (default 25%)
3

Add Your Students

In the "Add Student" section, enter each student's:

  • First Name
  • Last Name
  • Email (optional, for notifications)

Click "Add Student" after each entry. Students will appear in the roster list.

4

Get Your Class Code

After creating the roster, you'll see a Class Code (like "ABC123"). This is what students use to join your class.

üí° Pro Tip: Write this code on the board or share it via email/Google Classroom. Students need this exact code to join.
5

Organize Teams (Optional)

You can manually assign students to teams by:

  • Clicking the team color next to each student's name
  • Using "Shuffle Teams" to randomly assign
  • Teams will compete against each other in the war game

üìö Step 2: Creating Assignments

1

Access Assignment Library

Go to Assignments Library to see pre-made assignments or create your own.

2

Use Pre-Made Assignments

Browse the library for assignments like:

  • Literature: Romeo & Juliet, Macbeth, Hamlet, etc.
  • History: World Wars, Ancient Civilizations
  • Science: Biology, Chemistry, Physics
  • Math: Algebra, Geometry, Statistics

Click on any assignment to preview questions and assign to your roster.

3

Assign to Students

When you find an assignment you like:

  • Click "Assign to Roster"
  • Select your roster from the dropdown
  • Set point values for each question
  • Choose due date (optional)
üí° Pro Tip: Points earned from assignments automatically transfer to the war game. Students can spend these points on territories, weapons, and strategies.
4

Create Custom Assignments

To create your own assignments:

  • Click "Create New Assignment"
  • Add multiple-choice and short-answer questions
  • Set point values for each question
  • Save and assign to your rosters

üéÆ Step 3: Playing the War Game

1

Choose a Scenario

Go to War Game and select a scenario:

  • Business Mode: Corporate competition with departments
  • Historical Wars: World War II, Civil War, etc.
  • Literature: Romeo & Juliet, Macbeth, etc.
  • Pop Culture: Movies, music, sports
2

Select Your Roster

Choose the roster you want to play with. The game will:

  • Load all students from that roster
  • Assign them to teams based on your roster settings
  • Give them starting points (if they've earned any from assignments)
3

Students Join the Game

Students need to:

  • Go to Student War Game
  • Enter the class code you gave them
  • Select their name from the roster
  • Start playing!
4

How Students Play

Students can:

  • Buy Territories: Spend points to claim map territories
  • Attack Other Teams: Use weapons to take enemy territories
  • Form Alliances: Work with teammates strategically
  • Earn More Points: Complete assignments to get more resources
üí° Pro Tip: The team that controls the target percentage of territories (set in your roster) wins the game!
5

Monitor Progress

As the teacher, you can:

  • Watch the game unfold in real-time
  • See which students are most active
  • Check assignment completion rates
  • Adjust point values or game settings as needed

üìä Step 4: Tracking Student Progress

1

View Student Dashboards

In your roster management page, you can see:

  • Points Earned: Total points from assignments and quizzes
  • Territories Controlled: How many map territories each student owns
  • Assignment Status: Which assignments are completed
  • Team Performance: How teams are doing overall
2

Grade Assignments

For assignments with short-answer questions:

  • Go to the assignment in your roster
  • Click on individual student responses
  • Grade manually and award points
  • Points automatically transfer to the war game
3

Adjust Game Settings

You can modify the game at any time:

  • Change Target Percentage: Make the game easier or harder
  • Add Bonus Points: Reward students for good behavior
  • Reset Game: Start over if needed
  • Pause Game: Stop the game temporarily

üéØ Best Practices for Teachers

üìÖ Start Small

Begin with one roster and one assignment. Get comfortable with the system before expanding.

üèÜ Set Clear Goals

Tell students exactly how to win (e.g., "Control 25% of the map" or "Complete 3 assignments").

‚öñÔ∏è Balance Points

Make sure assignments give enough points for students to participate meaningfully in the war game.

üîÑ Regular Updates

Check progress regularly and provide feedback. Students love seeing their progress!

‚ö†Ô∏è Common Pitfalls:
  • Don't make assignments too hard - students need to earn points to play
  • Make sure students know their class code - write it clearly on the board
  • Start the war game after students have earned some points from assignments
  • Monitor for students who aren't participating and provide extra support
